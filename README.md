# RECEPIES BOOK
This is a Recepie book web app project made in Flask.

# INSTALL
Create a virtualenv and active it:
On Windows:
```
$ py -3 -m venv env
$ env\Scripts\activate.bat
```
On Linux:
```
$ python3 -m venv env
$ source env/bin/activate
```
### Project
```
$ pip install -e .
```
